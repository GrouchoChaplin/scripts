Purpose

The launcher is an interactive TUI to run presets, reconstructions, comparisons, and view dashboards and artifacts.

Main Capabilities

  • Preset selection from presets.d/
  • Analyzer and reconstructor runs with safety locks
  • Tree diff viewer using your chosen diff tool
  • Logs, dashboards, and Markdown export
  • Recent artifacts browser for JSON/CSV/HTML
