Analyzer — find_latest_flutter_source_vX.sh

Home Analyzer Reconstructor Launcher

Purpose

Scan backup roots, discover Dart files, and determine the newest instance per group, outputting CSV/JSON audit files.

Typical Command

find_latest_flutter_source_v1.9.2.sh \
  --paths "/backup/rootA" \
  --paths "/backup/rootB" \
  --pattern "*ir_imagery_tools*" \
  --group-by relpath \
  --include-instances \
  --csv dart_audit.csv \
  --json dart_audit.json \
  --log-file analyzer_run.log

Key Flags

  • --paths DIR – one or more backup roots.
  • --pattern PATTERN – subdirectory matcher (e.g. *ir_imagery_tools*).
  • --group-by – basename, relpath, or fullpath.
  • --include-instances – print all instances per group.
  • --csv / --json – structured audit exports.
  • --diff-vs-latest – compare each instance to the newest version.
