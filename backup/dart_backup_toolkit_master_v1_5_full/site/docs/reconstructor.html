Reconstructor — reconstruct_latest_tree_vX.sh

Home Analyzer Reconstructor Launcher

Purpose

Rebuild a synthetic Dart tree using analyzer CSV/JSON, in either latest-only or full-history mode.

Latest-only Example

reconstruct_latest_tree_v1.1.1.sh \
  --json dart_audit.json \
  --out latest_tree/ \
  --latest-only \
  --verify-checksums \
  --html-report latest_tree.html

Full-history Example

reconstruct_latest_tree_v1.1.1.sh \
  --csv dart_audit.csv \
  --out full_history_tree/ \
  --verify-checksums \
  --html-report full_history.html
