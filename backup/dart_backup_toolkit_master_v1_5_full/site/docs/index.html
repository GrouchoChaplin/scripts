Overview

This toolkit helps you reconcile Dart/Flutter source files across many backup roots, deterministically and reproducibly.

  • Analyzer – scans backup roots, finds Dart files, and determines the latest instance per group.
  • Reconstructor – rebuilds a synthetic tree (latest-only or full history) from analyzer output.
  • Comparer – compares reconstructed trees using your preferred diff tool.
  • Launcher – a TUI command center with presets, logs, dashboards, and safety locks.

Quick Start

./dart_backup_launcher.sh

Use the launcher menu to run analyzer, reconstructor, comparer, and view logs/dashboards.
